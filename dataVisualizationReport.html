Patient Data¶

By looking at the patient data, including their ages and genders, we can understand the trend and common demographic characteristics among patients in neurology. For example, very few young people are neurologic patients, especially people younger than 20. Patients older than 50 years old seem to be the majority. We can almost say the illness is somehow related to age. In terms of the gender, it is rather evenly distributed, but there are still more male patients than female.

Pupil Data¶

Pupil data measures different kinds of metrics including patients' pupil sizes, constriction velocity, dilation velocity, latency and NPi, etc. In this section, we look at data from both left and right eye of a patient and determine their linear relationship, which helps our data processing analysis in future sections. In the scatterplots below, by color coding each patient's diagnosis, we can understand its distribution more clearly. It is interesting to see circles with darker colors occupying more space than ones with lighter colors. It could mean that there are more people suffering Intraparenchymal Hemorrhage than people with Infection or other disease. Although not directly shown, people with Intraparenchymal Hemorrhage or Subarachnoid Hemorrhage seem to have lower constriction velocity and dilation velocity.

Diagnosis & Treatment¶

The two pie charts here in the diagnosis and treatment section, provide us with some insights about what are some common diagnosis among neurological patients. Diagnosis code 0, which represents Intraparenchymal Hemorrhage, takes more than 1/4 of the chart, whereas code number 8 (Intraventricular Hemorrhage) only has one occurence. On the second chart, more patients are treated with a combination of two types of interventions, but it's also not rare to see a combination of 3, including brain surgery.

In order to understand the relationship between the diagnosis and the treatment, we draw a bar chart to illustrate how the average number of treatments among patients differs according to the diagnosis. One extreme example here would be, for diagnosis code no.6 (Status Epilepticus), the average number of treatments is less than 0.25, while no.3 (Traumatic Brain Injury) is supposedly treated with a combination of two or more.

Surgery Data Analysis¶

This section allows us to explore the statistics of the surgeries. In the first bar chart, the number of patients undergoing a surgery and one that don't are compared. They are grouped based on their different NPi's. We can see that, non-surgery treatment is usually widely adopted, no matter what NPi a patient has. In the event of a patient with NPi higher than 4, they have a lower chance to having to go through a surgery. Of course, there are other factors involved in this decision.

In the next two charts, we further visualize how EVD and Hyperosmolar Therapy are used as intervention options other than surgery. The graphs show us that, for surgery patients, it's very likely that they are treated with a combination of EVD and Hyperosmolar Therapy. However, for non-surgery patients, especially those with higher NPi's, there is a big chance that they are treated with therapy only.

The below code constructs a regression model to predict the adoption of a surgery in treatment, using the data from NPi, Pupil size and Latency. The result proves to be 69% accurate, which is acceptable.

# Scale larger positive and values to between -1,1 depending on the largest
# Value in the data
min_max_scaler = preprocessing.MinMaxScaler(feature_range=(-1,1))
df = pd.read_csv("data2.csv", header=0).dropna()

# Clean up data
df.columns = ["NPiL","SizeL","LatL","NPiR","SizeR","LatR","Brain_Surgery_label"]

#x = df["label"].map(lambda x: float(x.rstrip(';')))

# Formats the input data into two arrays, one of independent variables
# and one of the dependant variable
X = df[["NPiL","SizeL","LatL","NPiR","SizeR","LatR"]]
X = np.array(X)
X = min_max_scaler.fit_transform(X)
Y = df["Brain_Surgery_label"]
Y = np.array(Y)

# Creating testing and training set into 20 and 80 percent
X_train,X_test,Y_train,Y_test = train_test_split(X,Y,test_size=0.20)

# Train scikit learn model 
clf = LogisticRegression()
clf.fit(X_train,Y_train)
print ('Accuracy of the model: {}'.format(clf.score(X_test,Y_test)))

Data Process - Ending NPi¶

Another data frame is constructed here in order to compare the changes or patients' NPi value. By processing data for a second time, we consider the changes in patients' pupil measurement. This time, we look at the start NPi and the ending NPi of each patient to discover the trend in those changes and how it's related to other data.

In [233]:
df2 = pd.DataFrame(columns=['ID','startNPi','endNPi','EVD','therapy','surgery'])
firstID = True
oldID = None
myRow = None

for index, row in data.iterrows():
    thisID = row['Study_ID']
    lastID = oldID
    
    lastRow = myRow
    thisRow = row
    
    if firstID:
        # Add the first row to the data frame
        start = row['NPiL']
        lastID = thisID
        firstID = False
    else:
        # When the ID is different and NPI is valid
        if thisID != lastID and row['NPiL']>0:
            end = lastRow['NPiL']
            if end>0:
                df2.loc[len(df2)] = [lastID, start, end,lastRow['EVD'],lastRow['Hyperosmolar.Therapy'],lastRow['Brain.Surgery']]
            start = row['NPiL']
            lastID = thisID
    myRow = row
    oldID = thisID
    
end = myRow['NPiL']
df2.loc[len(df2)] = [oldID, start, end,lastRow['EVD'],lastRow['Hyperosmolar.Therapy'],lastRow['Brain.Surgery']]

# Convert ID column to int type
df2 = df2.astype({"ID": int,"therapy": int,"EVD": int,"surgery": int})

NPi Changes¶

In this last section, we first divide patient data into groups based on what intervention they are given. The graphs are color coded, where red means their NPi drops and green means it increases. Each vertical line represents one patient and the length of the line is how much their NPi has changed.

By comparing these three graphs, we can tell the surgery usually causes more drastic changes, whereas on the other two charts, dots appear more than long lines. If we consider removing extreme cases, therapy seems to affect NPi value in a more positive way than the others.
